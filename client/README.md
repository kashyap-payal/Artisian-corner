# 🛍️ Artisans Corner

A full-stack MERN E-Commerce Marketplace where Sellers can upload products and Buyers can purchase and review them.

---

## 🚀 Features

- User Authentication (JWT)
- Buyer, Seller & Admin Roles
- Product Management
- Shopping Cart
- Checkout
- Order Management
- Product Reviews
- Cloudinary Image Upload
- Admin Dashboard

---

## 🛠️ Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Cloudinary
- JWT Authentication
